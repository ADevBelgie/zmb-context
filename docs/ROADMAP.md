# ZMB Public Roadmap

**Mission:** To build the ultimate context architect for AI coding assistants.

## Phase 1: Foundation (Complete)
- [x] **Core CLI:** `zmb-context` tool for generating context files.
- [x] **Smart Focus:** Analyze specific parts of the codebase (`--focus`).
- [x] **.cursorrules Generation:** Automatically create rules for Cursor IDE.
- [x] **PyPI Deployment:** Available via `pip install zmb-context`.

## Phase 2: Refinement (Current)
- [x] **Rich UI:** Beautiful terminal output.
- [x] **Interactive Mode:** Guided context generation.
- [ ] **Expanded Focus Modes:** Support for more languages and frameworks.

## Phase 3: ZMB Pro (Upcoming)
- [ ] **Team Context Sync:** Share context definitions across your team.
- [ ] **Cloud Memory:** Persistent context storage for distributed teams.
- [ ] **IDE Plugins:** Direct integration into VS Code and JetBrains.
- [ ] **Analytics:** Track token usage and context effectiveness.

> [!NOTE]
> This roadmap is subject to change based on user feedback and market trends.
